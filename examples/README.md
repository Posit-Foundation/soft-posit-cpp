# SoftPosit Examples

This directory contains example applications demonstrating the use of the SoftPosit library.

## Available Examples

### Mandelbrot Set

- Demonstrates the use of posits in fractal generation
- Shows the difference between standard floating-point and posit arithmetic
- Generates PPM format images of the Mandelbrot set

## Building the Examples

### Prerequisites

- CMake (version 3.10 or higher)
- C++17 compatible compiler
- The SoftPosit library installed on your system

### Step 1: Build and Install the SoftPosit Library

First, build and install the SoftPosit library:

```bash
# Create build directory
mkdir build
cd build

# Configure with CMake
cmake .. -DCMAKE_INSTALL_PREFIX=../install -DBUILD_SHARED_LIBS=ON

# Build and install
make
make install
```

### Step 2: Build the Examples

Navigate to the example directory and build:

```bash
# Go to the mandelbrot example directory
cd examples/mandelbrot

# Create build directory
mkdir build
cd build

# Configure with CMake
cmake .. -DCMAKE_PREFIX_PATH=../../install

# Build the examples
make
```

This will create two executables:

- `mandelbrot_dynamic`: The standard double-precision implementation
- `posit_mandelbrot_dynamic`: The p16 posit-based implementation

### Step 3: Run the Examples

Before running the examples, ensure the library path is set:

```bash
# On Linux/macOS
export LD_LIBRARY_PATH=../../install/lib:$LD_LIBRARY_PATH

# On macOS (alternative)
export DYLD_LIBRARY_PATH=../../install/lib:$DYLD_LIBRARY_PATH
```

Then run either of the executables:

```bash
# Run the standard implementation
./mandelbrot_dynamic

# Run the posit implementation
./posit_mandelbrot_dynamic
```

Both implementations will generate PPM format images:

- `mandelbrot.ppm`: Generated by the standard implementation
- `posit_mandelbrot.ppm`: Generated by the posit implementation

## Viewing the Results

The generated PPM images can be viewed using:

- ImageMagick (`display` command)
- GIMP
- Most web browsers

## Troubleshooting

If you encounter issues:

1. Ensure the SoftPosit library is installed and findable by CMake
2. Verify that your compiler supports C++17
3. Check that the SoftPosit library was built with shared libraries enabled
4. Make sure the library path is in your system's library search path
5. If you get "library not found" errors, verify that:
   - The library was built with `-DBUILD_SHARED_LIBS=ON`
   - The library is installed in the correct location
   - The library path is properly set in your environment
